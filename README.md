# 🛒 Amazon Clone

A simple **Amazon homepage clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of Amazon's landing page for learning and practice.

## ✨ Features

- Responsive Design
- Amazon-inspired UI
- Navigation Bar
- Hero Section
- Product Cards
- Footer

## 🛠️ Tech Stack

- HTML5
- CSS3

## 🚀 Getting Started

```bash
git clone https://github.com/bhudev0905/amazon-clone.git
cd amazon-clone
```

Open `index.html` in your preferred web browser.
